Print

The print statement is one of the most simple Python commands:
#Input:

print("Hello World")
This code simply returns:
#Output:

Hello World
You can also add variables to your print statements. For instance:
#Input:

x = 2 + 3
print("2 + 3 = " + x)
Returns:
#Output:

2 + 3 = 5
Finally, you can also create an f-string, putting your variables into brackets instead of using the + sign. This is especially useful when you have a lot of variables in one line:
#Input:

y = 2 z = 3 x = y + z
print(f"{y} + {z} = {x}")
Returns:
#Output:

2 + 3 = 5

Input

The input function is relatively similar to the print function, but it asks the user for an answer:
#Input:

input("What is your favorite color: ")
This will return the same message, but the user can enter answer:
#Output:

What is your favorite color: Blue
You can also store a user's input as a variable, then use it later:
#Input:

userName = input("What is your name: ")
print(f"Hello, {userName}!")
This returns:
#Output:

What is your name: Bob
Hello, Bob!

Type

The type function simply returns the type of an object. Objects can have several different types, and here are some of the most common:
Type Abbreviation Explanation Examples
String str Can be anything but will not possess any special qualities. For instance, one cannot add '1' and '2' if they are both strings; they must be converted to integers. 'Hello World', '1', '0.3'
Integer int Any whole number; can be added, multiplied, etc. in equations -5, 0, 100
Float float Any decimal; possesses the same characteristics as int -6.12, 0.324, 3.1415
List list A group of objects contained in square brackets and separated by commas ['Hello', 9, 0.1, True]
Dictionary dict Similar to a list but organizes elements in key:value pairs; also uses curly brackets. {'First':'Bob', 'Last':'Smith', 'Age':35}
You can simply find the type of an object by using type():
#Input:

type('Hello')
This returns:
#Output:

str

For Loops

Finally, for loops are the best way to complete repeated tasks with few lines of code:
#Input:

for i in range(5): 
    print(i)
This returns:
#Output:
1
2
3
4
5
For loops can be used to loop through different values. You can also use if statements when writing for loops:
#Input:
for i in range(15): 
    if i % 5 == 0:
        print(i)
#Output:
5
10
15
